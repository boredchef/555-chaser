# 555-chaser
I made this heart shaped led chaser PCB from the start project tutorial on Stasis. How it works is that it uses 2 different ICs which is the 555 timer and the 4017 IC. The 555 component basically is the brains to make the leds blink on and off using astable mode which produces an signal that swithces from on and off. Since we are using a lot of LEDs there is not enough pins on the 555 timer to wire them all so that is why we use the 4017 IC which has the pins. 

This PCB when connected to power and when the potentiometer is turned to let power through the components which then tell the leds to turn on and off. I used this starter project since I thought it would be really fun to learn how to design pcbs using software instead of having to wire things individually on a breadboard and be able to see something from CAD software come to life in hardware which is pretty sick, I also added hearts to gift to someone special :)
